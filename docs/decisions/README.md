# Architecture Decision Records

ADRs registram decisões arquiteturais duradouras, seu contexto e consequências. Estados possíveis:
`Proposto`, `Aceito`, `Substituído` e `Rejeitado`. Uma decisão aceita não é editada para apagar o
passado; mudanças relevantes criam um novo ADR que referencia o anterior.
