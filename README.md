VisualnsCode is an open-source desktop IDE that puts Claude, GPT, Gemini, Ollama and your favorite
dev tools in one focused workspace. Simple mode for beginners. Full power for pros.

## Install

**macOS & Linux ‚ÄĒ one line:**

```bash
curl -fsSL https://raw.githubusercontent.com/spxmiguel/visualnscode/main/scripts/install.sh | bash
```

**Windows ‚ÄĒ PowerShell or cmd:**

```powershell
irm https://raw.githubusercontent.com/spxmiguel/visualnscode/main/scripts/install.ps1 | iex
```

After install, type **`spxcode`** in any terminal to open the app.

> Download installers manually from [Releases](https://github.com/spxmiguel/visualnscode/releases):
> `.pkg` (macOS), `.AppImage` / `.deb` (Linux), `.msi` (Windows).

## What it does

|                             |                                                                                                                               |
| --------------------------- | ----------------------------------------------------------------------------------------------------------------------------- |
| ūü§Ė **Multi-provider AI**    | Switch between Claude, GPT-4o, Gemini, Ollama, OpenRouter and any OpenAI-compatible endpoint without touching a line of code. |
| ūüõ† **Agent teams**           | Build workflows with specialized agents ‚ÄĒ Architect, Frontend Dev, Reviewer, Tester ‚ÄĒ running in sequence or parallel.        |
| ūüéõ **Two modes**             | Simple mode shows only what matters. Advanced mode reveals terminal, Git, diffs, logs, permissions, and agent controls.       |
| ūüĒí **Safe by default**      | AI never edits files silently. Every change goes through diff ‚Üí approve ‚Üí checkpoint ‚Üí apply.                                 |
| ūüöÄ **Integrated deploy**    | Vercel, Firebase, Supabase, GitHub Pages ‚ÄĒ from build to production URL with explicit confirmation.                           |
| ūüõ† **Auto-onboarding**       | Detects Git, Node, pnpm, CLIs, and AI providers already installed. Guides through setup in plain language.                    |
| ūüßĪ **Guided projects**      | Describe an idea, review the suggested stack, then create from 13 versioned templates with explicit external-action consent.  |
| ūüĆŅ **Git without friction** | Save versions simply or use branches, tags, merges, conflicts, GitHub issues, PRs, Actions, and releases.                     |

## Development

**Prerequisites:** Node.js ‚Č• 20.18, pnpm ‚Č• 9, Git.

```bash
# clone
git clone https://github.com/spxmiguel/visualnscode.git
cd visualnscode

# install all workspace dependencies
pnpm install

# run Electron app in dev mode
pnpm dev

# run landing page only
pnpm dev:landing

# checks
pnpm lint && pnpm typecheck && pnpm test
pnpm test:e2e && pnpm test:lighthouse

# build everything
pnpm build
```

## Packaging

```bash
# inside apps/desktop ‚ÄĒ builds for the current platform
pnpm --filter @visualnscode/desktop release

# per platform
pnpm --filter @visualnscode/desktop release:mac
pnpm --filter @visualnscode/desktop release:linux
pnpm --filter @visualnscode/desktop release:win
```

Artifacts land in `release/`. Cross-platform builds run automatically via GitHub Actions on every
tagged release.
